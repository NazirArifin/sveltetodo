# Sveltekit Todo App

### Plugin

- Sveltekit
- Tailwindcss
- Preline

### Clone the repository
```bash
git clone git@github.com:NazirArifin/sveltetodo.git
```

### Gunakan degit
```bash
npx degit git@github.com:NazirArifin/sveltetodo
```

### Install dependencies
```bash
npm install
```

### Run the project
```bash
npm run dev
```
